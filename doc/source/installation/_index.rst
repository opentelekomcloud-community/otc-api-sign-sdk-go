Installation
===============

.. toctree::
   :maxdepth: 3
   :includehidden:

The latest state of the module can be installed directly from the GitHub repository.

Installation using pip:
------------------------------

.. code-block:: shell
   :caption: go install

   go get -u github.com/opentelekomcloud-community/otc-api-sign-sdk-go@latest

.. code-block:: go
   :caption: go.mod

    module your/module/name
    go 1.17

    require github.com/opentelekomcloud-community/otc-api-sign-sdk-go v0.0.0-00010101000000-000000000000
